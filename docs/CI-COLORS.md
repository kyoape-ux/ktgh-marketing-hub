# 光田綜合醫院 CI 色票規範

## 官方主色系

| 名稱   | HEX       | RGB               | 用途科別                    |
|--------|-----------|-------------------|----------------------------|
| 深綠   | `#006341` | 0, 99, 65         | 院本色 / Dashboard 側欄     |
| 生命綠 | `#6EC300` | 110, 195, 0       | 肝膽腸胃、耳鼻喉、點綴用    |
| 寬廣藍 | `#00AAC8` | 0, 170, 200       | 健管、達文西、神經內科      |
| 寧靜靛 | `#14286E` | 20, 40, 110       | 睡眠中心、骨科              |
| 陽光黃 | `#FFB900` | 255, 185, 0       | 高壓氧、社區、中醫          |
| 大地棕 | `#AA7D5F` | 170, 125, 95      | 癌症、失智、骨鬆            |
| 紅（降飽和） | `#C0313D` | 192, 49, 61  | 醫美中心                   |

## Dashboard UI 應用規則

```
側欄底色       →  深綠  #006341
Active nav dot  →  生命綠 #6EC300（亮一階，做點綴）
主按鈕背景     →  深綠  #006341
次要按鈕背景   →  #EAF5EE（深綠淡化版）
互動 KPI 線    →  寬廣藍 #00AAC8
頁面底色       →  #F3F9F6（帶綠調的暖白）
```

## 不可使用
- 黑色 (#000000) 作為按鈕背景
- 純紫色系（非 CI 色）
- 高飽和螢光色
